# Smart Class Scheduler (SIH Final)

Final deliverable (Flask + SQLite) for Smart India Hackathon-style project.

## Features
- Admin login (default admin/admin123)
- Add Teachers, Subjects (with weekly demand), Rooms
- Generate Timetable (Greedy)
- AI Optimize (heuristic balancing)
- Timetable calendar view

## Run locally
1. Create virtual env and activate:
   python -m venv venv
   source venv/bin/activate   (Linux/Mac) or venv\Scripts\activate (Windows)

2. Install requirements:
   pip install -r requirements.txt

3. Run:
   python app.py

4. Open: http://127.0.0.1:5000
